# Rhymix YouTube Content Widget

## Overview
Set a YouTube channel ID to fetch the latest videos from its RSS feed and display them in Rhymix.

## Installation
* Path: `./widgets/youtube_content`
* Steps:
```bash
cd /path/to/rhymix/widgets
git clone https://github.com/seungh/rhymix-youtube-content-widget youtube_content
```

## Configuration
* Channel ID (required): Enter the YouTube channel ID.
* Widget title: If empty, the channel name is shown.
* Max videos: Number of videos to display.
* Sort order: Recent / Views / Likes / Random.
* Cache TTL (minutes): How long to cache RSS results.
* Open in new tab: Open links in a new tab.
