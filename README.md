# Rhymix YouTube Content Widget

## 소개
유튜브 채널 ID 값을 설정하면, 해당 채널의 RSS 피드에서 최신 영상을 가져와 라이믹스에서 보여주는 위젯입니다.

## 설치방법
* 설치경로: `./widgets/youtube_content`
* 설치방법:
```bash
cd /path/to/rhymix/widgets
git clone https://github.com/seungh/rhymix-youtube-content-widget youtube_content
```

## 설정방법
* 채널 ID (필수): 유튜브 채널 ID를 입력합니다. 
* 위젯 제목: 비워두면 채널명이 표시됩니다.
* 최대 표시 개수: 표시할 영상 개수를 지정합니다.
* 정렬 순서: 최근/조회수/좋아요/랜덤 정렬을 선택합니다.
* 캐시 유지 시간(분): RSS 결과를 캐시에 저장하는 시간입니다.
* 새 탭으로 열기: 링크를 새 탭에서 열지 여부를 설정합니다.
